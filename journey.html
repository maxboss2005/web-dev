1. INTRODUCTION TO WEB DEVELOPMENT

What is Web Development?

Web development is the process of creating websites or web applications that are accessed via the internet.


Types of Web Development

1. Front-end Development: Involves what the users see. (HTML, CSS, JAVASCRIPT)
2. Back-end Development: Involves what powers the site behind the scenes. (Python, Node.js, PHP)
3. Full-stack Development: A combination of both front-end and back-end development.


PRACTICE

Open the code editor on your pc (Visual Studio Code preferred)
Create a file called intro.html.
Input code below. You can customize it.


                <!DOCTYPE html>
                <html>
                <head>
                    <title>
                    My First Web Page
                    </title>
                </head>
                <body>
                    <h1>
                    Hello World!
                    </h1>
                    <p>
                    I am learning web development.
                    </p>
                </body>
                </html>

2. HTML - STRUCTURE OF WEB PAGES

HTML (HyperText Markup Language) is the skeleton of a webpage.


Basic Tags

- < html >, < head >, < title >, < body >.
- Headings: < h1 > to < h6 >.
- Paragraphs: < p >
- Links: < a href=" " >
- Images: < img src=" " >
- Lists: < ul >, < ol >, < il >
- Line break: < br >
- Bold (font-weight in css): < strong > or < b >
Italic (font-style in css): < em > or < i >
Underline (text-decoration in css): < u >


PRACTICE

Create a file called structure.html
Create a simple page about yourself using headings, images, and lists.

3. CSS - STYLING THE WEB

CSS (Cascading Style Sheets) is used to design and layout web pages.


Ways to Add CSS:

- Inline (style=" ")
- Internal (< style > in the < head >)
- External (style.css file)


Basic Properties

Color, font-size, background-color, padding, margin, border.


Practice:

Create style.css and link it to your structure.html file. 


                    body {
                        background-color: #f4f4f4;
                        font-family: Arial, sans-serif;
                        color: #333;
                    }

                    h1 {
                        color: royalblue;
                    }

4. JAVA - MAKING PAGES INTERACTIVE

JavaScript is a programming language used to add interactivity (e.g clicking, animations, form validation).


Examples:

- Change text on click.
- Show/hide content.
- Alert boxes.


Practice

Add this to your HTML file: 


< button onclick="sayHello()">
Click Me
< /button >
< script >
function sayHello() {
alert("Hello, User1");
}
< /script >

5. FORMS AND INPUT

Forms let users submit data (contact, sign-up, feedback).


Common Form Tags:

- < form >, < input >, < textarea >, < button >, < label >


Practice:

Create a file form.html 


                    < form >
                    < label > 
                    Name: 
                    < /label > < br >
                    < input type="text" required > < br >

                    < label >
                    Email: 
                    < /label > < br >

                    < input type="submit" value="Send" >

                    < /form >

6. LAYOUT WITH FLEXBOX & GRID

Modern layout techniques using CSS:

- Flexbox : One-dimensional layout (horizontal or vertical)
- Grid : Two-dimensional layout (rows and columns)


Practice:

Try this Flexbox layout: 


                    < div  style ="display: flex; gap:
                     10px;">
                        < div style="background: lightblue;
                     padding: 20px;"> Box 1 < /div >
                        < div style="background: lightcoral;
                    padding: 20px;"> Box 1 < /div >
                    </div >

7. RESPONSIVE DESIGN & MEDIA QUERIES

Make your website look good on phones and tablets using media queries.


Practice: 


                        @media (max-width: 600px) {
                            body {
                                background-color: lightyellow;
                            }
                        }
